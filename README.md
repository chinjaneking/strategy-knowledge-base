# 📚 谋策智能体独立知识库 (Strategy Knowledge Base)

**版本**: 2.1.0
**定位**: 谋策智能体的独立智库模块，支持热更新和独立维护

---

## 🎯 项目简介

这是一个独立的 Python 包，为谋策智能体提供完整的知识体系支持。将智库从主项目中剥离，实现：

- ✅ **独立维护** - 知识内容升级无需修改主代码
- ✅ **热更新** - 运行时动态重载知识库
- ✅ **版本管理** - 知识库独立版本号，清晰追踪变更
- ✅ **便捷扩展** - 通过 JSON 文件即可补充新谋略

## 🚀 安装使用

### 安装方式

#### 方式1：本地安装（开发模式）

```bash
git clone https://github.com/chinjaneking/strategy-knowledge-base.git
cd strategy-knowledge-base
pip install -e .
```

#### 方式2：作为子模块添加到主项目

```bash
# 在主项目中添加子模块
git submodule add https://github.com/chinjaneking/strategy-knowledge-base.git strategy_knowledge_base

# 安装
pip install -e ./strategy_knowledge_base
```

#### 方式3：直接引用（无需安装）

```python
import sys
sys.path.insert(0, '/path/to/strategy-knowledge-base')
from strategy_knowledge import KnowledgeManager, WisdomManager
```

## 📖 使用指南

### 1. 静态知识库使用

```python
from strategy_knowledge import KnowledgeManager

# 创建管理器
km = KnowledgeManager()

# 获取知识库版本
print(km.get_version())  # 2.1.0

# 列出所有知识来源
sources = km.list_sources()
print(sources)
# ['三十六计', '鬼谷子', '孙子兵法', '毛泽东思想', '资治通鉴']

# 获取指定来源信息
info = km.get_source_info("孙子兵法")
print(info)
# {
#   'name': '孙子兵法',
#   'description': '兵学圣典，战略战术的至高智慧',
#   'source': '《孙子兵法》',
#   'category': '战略战术',
#   'tactics_count': 5
# }

# 获取指定谋略详情
tactic = km.get_tactic("孙子兵法", "虚实篇")
print(tactic)
# {
#   'definition': '掌握主动权，避实击虚',
#   'application': ['资源分配', '竞争策略', '主动权的掌握'],
#   'quote': '夫兵形象水...',
#   'details': '分散对方力量，集中己方优势...'
# }

# 搜索谋略
results = km.search_tactics("危机")
for r in results:
    print(f"{r['source']} - {r['name']}")

# 获取系统提示词
system_prompt = km.get_system_prompt()
```

### 2. 动态智慧库使用

```python
from strategy_knowledge import WisdomManager

# 创建管理器
wm = WisdomManager()
# 数据默认保存在 ~/.strategy_knowledge/wisdom_library.json

# 添加智慧
wisdom_id = wm.add_wisdom(
    original_question="面临竞品价格战，该如何应对？",
    action_taken="选择差异化路线，避开正面竞争",
    final_result="成功保住利润率，市场份额稳定在15%",
    extracted_wisdom="弱势面对强势时，避实击虚是生存之道，不可正面硬拼",
    tags=["价格战", "差异化", "生存策略"],
    category="商业竞争"
)
print(f"智慧已保存，ID: {wisdom_id}")

# 获取相关智慧（供智能体使用）
relevant = wm.get_relevant_wisdom("竞争对手降价了怎么办？", limit=3)
print(relevant)

# 搜索智慧
results = wm.search_wisdoms("差异化")
for r in results:
    print(f"[{r['category']}] {r['extracted_wisdom'][:50]}...")

# 获取统计信息
stats = wm.get_statistics()
print(f"总计: {stats['total_count']} 条智慧")
print(f"分类: {stats['categories']}")

# 导出智慧库
wm.export_to_file("./my_wisdom_backup.json")

# 导入智慧库（合并模式）
wm.import_from_file("./shared_wisdom.json", merge=True)
```

### 3. 知识库热更新

```python
from strategy_knowledge import KnowledgeManager

km = KnowledgeManager()

# 编辑 static_library.json 文件后，重新加载
km.reload()

# 获取更新后的内容
print(km.get_version())
```

## 🔧 升级知识库

### 添加新谋略

编辑 `strategy_knowledge/static_library.json`：

```json
{
  "knowledge_base": {
    "孙子兵法": {
      "core_tactics": {
        "新增谋略名称": {
          "definition": "定义",
          "application": ["应用场景1", "场景2"],
          "quote": "原文引用",
          "details": "详细说明"
        }
      }
    }
  }
}
```

### 通过代码添加

```python
from strategy_knowledge import KnowledgeManager

km = KnowledgeManager()
km.add_tactic("孙子兵法", "新增谋略", {
    "definition": "...",
    "application": ["..."],
    "quote": "...",
    "details": "..."
})

# 重新加载以生效
km.reload()
```

## 🔗 与主项目集成

### 主项目修改方案

1. **添加依赖**到 `requirements.txt`：
```
-e ./strategy_knowledge_base  # 如果是子模块
# 或
strategy-knowledge-base>=2.1.0  # 如果发布到PyPI
```

2. **修改主代码**，替换原有知识库引用：

```python
# 原代码（knowledge_base.py 内嵌）
from core.knowledge_base import get_system_prompt, STRATEGY_KNOWLEDGE_BASE

# 新代码（独立包）
from strategy_knowledge import KnowledgeManager, WisdomManager

class StrategyAgent:
    def __init__(self):
        self.knowledge = KnowledgeManager()
        self.wisdom = WisdomManager()

    def get_system_prompt(self):
        return self.knowledge.get_system_prompt()
```
